# Platform Availability SLO

## Service Description
The platform provides a Kubernetes-based execution environment for
business-critical workloads.

## SLO
- Availability: 99.95% per calendar month

## SLIs
- Kubernetes API availability
- Successful workload scheduling

## Error Budget
- 0.05% unavailability per month

## Error Budget Policy
- Excessive burn pauses non-critical changes
- Incidents trigger error budget review
